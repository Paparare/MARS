# Data Directory

Place your GPQA dataset files here:

- `gpqa_diamond_train.csv` - Training split
- `gpqa_diamond_test.csv` - Test split

## Dataset Format

The GPQA CSV files should have the following columns:
- `Question` - The question text
- `Correct Answer` - The correct answer text
- `Incorrect Answer 1` - First incorrect answer option
- `Incorrect Answer 2` - Second incorrect answer option
- `Incorrect Answer 3` - Third incorrect answer option
- `High-level domain` - Category (Physics, Chemistry, Biology)
- `Subdomain` - More specific topic area
